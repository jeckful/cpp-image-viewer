# C++ Image Viewer – University Project

A C++ application for reading, editing, and displaying PPM images, developed as part of a university group project course. The viewer uses SDL2 for graphical rendering.

---

## Project Structure

```
cpp-image-viewer/
├── bin/                         # Compiled executables, generated by make
├── build/                       # CMake build artifacts
├── data/                        # Sample PPM images
│   ├── image1.ppm
│   ├── image2.ppm
│   └── image3.ppm
├── src/                         # C++ source code
│   ├── Image.cpp / .h           # Image class (load, save, edit)
│   ├── Pixel.h                  # Pixel structure (RGB)
│   ├── ImageViewer.cpp / .h     # SDL2-based image viewer
│   ├── mainTest.cpp             # Regression tests
│   ├── mainExemple.cpp          # Usage examples
│   └── mainAffichage.cpp        # Graphical viewer entry point
├── Makefile                     # Top-level build wrapper
├── CMakeLists.txt
├── .gitignore
└── README.md
```

---

## Build & Run

**Requirements**
- C++11 compatible compiler (e.g. `g++`)
- CMake (v3.0+)
- SDL2 development libraries
- Make

**Commands** (from the project root)
```bash
make                # Compile all targets
make clean          # Delete build/ and bin/
```

---

## Executables

**test** — Runs regression tests on the `Image` class.
```bash
./bin/test
```

**exemple** — Demonstrates how to create, edit, and save images programmatically.
```bash
./bin/exemple
```

**affichage** — Opens and displays a PPM image using SDL2.
```bash
./bin/affichage
```

---

## Features

**Image manipulation**
- Read and write PPM images (P3 format)
- Edit individual pixels
- Clear image with a solid color
- Draw filled rectangles

**Display**
- SDL2 graphical viewer with zoom (T / G keys)
- Console output mode for debugging

---

## Technical Details

- **Language**: C++11
- **Image format**: PPM (P3, ASCII)
- **GUI Framework**: SDL2
- **Build system**: CMake
